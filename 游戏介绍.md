# 蝗灾农场防御战游戏介绍

## 
《蝗灾农场防御战》是一款融合了**地图探索、回合制卡牌战斗、农场养成**的轻度肉鸽游戏。玩家将一边抵御虫灾，一边经营农田，在每一轮战斗与种植中获取相应资源，保卫好你的家园！

## 游戏背景
蝗灾来袭，农场岌岌可危。玩家需要在未知地图中不断前进，面对普通敌人与首领挑战；同时通过种植、收获、作物效果与资源管理，逐步强化自己的卡组和角色能力，撑过一层又一层危机。

## 核心玩法循环
1. 地图探索：进入新楼层后在网格地图中移动，触发不同类型房间事件。
2. 战斗对抗：在战斗房与敌人进行回合制卡牌战斗，玩家需要根据战斗实时状态，合理利用好手牌，击败遇到的敌人。
3. 战后成长：获得金币、物资或其他收益，可以升级手中卡牌或获取战斗增益，优化下一场战斗准备。
4. 农场经营：在农场种植作物，等待生长并收获卡牌奖励与增益效果。
5. 层间抉择：完成楼层后进行新的选择与配置，进入下一轮挑战。

## 主要系统特色
- 地图系统：楼层化随机地图，包含战斗、奖励、陷阱与 Boss 房等内容。
- 战斗系统：回合制卡牌玩法，围绕抽牌、出牌、状态效果与回合节奏展开。
- 战前构筑调节：玩家可以在战前通过消耗农作物，进行不带牌和加牌，避免杀戮尖塔中的删牌困难难以无限的问题，让游玩者可以比较简单的感受到回合制游戏的乐趣，以此达到安利杀戮尖塔的目的。🤓
- 农场系统：多地块种植、生长、收获，形成与战斗联动的养成路径。
- 作物效果系统：收获后可带来属性变化或策略增益，影响后续探索与战斗。
- 末影箱/奖励机制：阶段性提供新的成长选择，强化肉鸽局内构筑体验。
- 存档系统：支持保存与读取当前进度，便于持续体验和调试。

## 当前版本说明
当前版本已具备完整的主循环框架（探索、战斗、农场、成长、保存）。
后续将继续扩展更多敌人、卡牌、作物与事件内容，提升策略深度和重复游玩价值。

## 后续版本扩展亮点

### 1. 主题化赛季内容
- 后续版本计划推出“蝗灾季”“寒潮季”“旱灾季”等主题赛季，每个赛季包含独立敌人生态、天气机制与限定作物，进一步提升游戏的趣味性与挑战性。

### 2. 增加动态天气与地块生态系统
- 根据主题化赛季系统新增天气系统（暴雨、干旱、虫潮、霜冻），实时影响作物成长、战斗状态和地图事件概率。
- 引入地块肥力与连作惩罚机制，让玩家在“短期收益”和“长期规划”之间做决策。

### 3. 升级美术与动画效果
- 完善角色、敌人、作物与场景美术风格统一性，提升项目整体完成度与视觉辨识度。
- 增加战斗关键动作动画（出牌、受击、击败、状态触发）与UI动态反馈，提升操作爽感和观感表达。

### 4. 完善新手教学关卡
- 新增分步式教学关卡，逐步引导地图移动、出牌逻辑、农场管理与战前构筑。
- 通过“示例回合 + 实战演练”帮助玩家快速理解策略核心，缩短学习曲线，提升留存与安利效果。

### 5. 进阶构筑系统。
- 在现有战前调牌基础上，完善“消耗资源进行不带牌和加牌”的机制，并与农作物产出形成闭环。
- 扩展作物与牌组机制，让玩家在资源管理与战斗强度之间做更有策略性的取舍。
- 根据牌组与作物效果衍生多种游戏玩法，比如在农场攒足够资源然后无脑平推的轮椅流，又比如不携带多余初始资源，尽量在普通关和低难度boss关省资源的高手玩法，鼓励玩家自助探索更多新奇玩法。

### 6. 增加难度分层与挑战模式
- 增加更高难度选项（敌人数值增强、稀有事件更频繁、资源更紧缺），提供可持续挑战目标。
- 引入挑战词条与周目加成机制，鼓励玩家多轮尝试，形成稳定的高复玩循环。

### 7. 完善游戏剧情
- 每层 Boss 绑定独立剧情片段与行为模式，形成“战斗机制 + 世界观推进”的双重反馈。

## 运行环境与启动说明

### 运行环境要求
- 操作系统：Windows 10/11（64 位）
- 必需组件：`.NET 8.0`（建议安装 `.NET Desktop Runtime 8.0 x64`）

### 首次运行步骤
1. 在 Release 页面下载 `FarmRoguelike-Windows.zip`。
2. 将压缩包完整解压到同一个文件夹（不要只拷出 `.exe` 单文件运行）。
3. 确认 `FarmRoguelike.exe` 与 `data_新建游戏项目_windows_x86_64` 文件夹在同级目录。
4. 双击 `FarmRoguelike.exe` 启动游戏。

### 若提示缺少 .NET
- 安装 `.NET 8.0 Desktop Runtime x64` 后重新启动游戏。
- 安装地址：`https://dotnet.microsoft.com/download/dotnet/8.0`
