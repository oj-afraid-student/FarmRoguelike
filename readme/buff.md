# 农场肉鸽 (FarmRoguelike) - 状态效果说明

本文档记录了游戏内目前实装的所有 Buff（增益）和 Debuff（减益）的详细效果机制。

## 🔴 减益状态 (Debuff)

| 状态名称 | 核心机制 | 衰减与叠加规则 |
| :--- | :--- | :--- |
| **中毒** (Poison) | **持续真实伤害**。在回合开始时，受到相当于 `目标最大生命值 4% × 中毒层数` 的真实伤害。（非常克制高血量 Boss） | **无限期叠加，从不衰减**。只要挂上就会一直掉血。 |
| **燃烧** (Burn) | **爆发真实伤害**。在回合结束时，受到相当于 `目标最大生命值 8% × 燃烧层数` 的真实伤害。 | **可叠加**，但每回合结束结算后，层数自动 **-1**。 |
| **眩晕** (Stun) | **硬控**。被眩晕的目标将**必定跳过**下一个回合的行动。 | 行动被跳过一次后，自动消耗并解除该状态。 |
| **冰冻** (Freeze) | **软控**。回合开始时，有 **20% 的概率**被彻底冻住从而跳过本回合行动。 | **独立概率看脸**。如果判定触发了跳过回合，冰冻状态才会解除。 |
| **破甲/易伤** (Vulnerable) | 受到的任何攻击伤害额外**增加 20%**。 | **可叠加**，最多叠 2 层（最高承伤+40%），有持续回合限制。 |
| **虚弱** (Weak) | 造成的任何攻击伤害**降低 10%**。 | **可叠加**，最多叠 3 层（最高降伤 30%），有持续回合限制。 |
| **诅咒** (Curse) | 每当你**打出一张牌**时，便会受到 `2 × 诅咒层数` 的伤害。 | 通常为陷阱或特定敌人施加。 |
| **缴械** (Disarm) | **受到此效果者在持续期间内无法使用任何攻击类型的牌。** | **不可叠加但可刷新时间**，时间按回合计算。 |
| **沉默** (Silence) | **受到此效果者在持续期间内无法使用任何技能/能力类型的牌。** | **不可叠加但可刷新时间**，时间按回合计算。 |
| **和平** (Pacifist) | **本回合内绝不可造成任何攻击伤害**，为正面大回复技能的附带惩罚。 | **当前回合结束移除**。 |

## 🟢 增益状态 (Buff)

| 状态名称 | 核心机制 | 衰减与叠加规则 |
| :--- | :--- | :--- |
| **力量** (Strength) | 每次使用攻击牌造成的最终伤害**提升 3 点**。 | **可叠加**，最多叠加 5 层（最多增加 15 点全技能平伤）。 |
| **精准** (Precision) | **绝对真实伤害**。拥有此状态时，你使用所有的攻击牌都将**完全无视敌方的护甲值**，直接扣除其真实血量。 | 持续时间通常为本回合。不可叠加层数。 |
| **嗜血** (Lifesteal) | **吸血续航**。拥有此状态期间，你对敌人造成的所有实际伤害的 **20%** 会转化为自身的生命回复。 | 持续回合制，不可叠加层数。 |
| **活力** (Vigor) | **持续恢复**。在每一回合开始时，自动恢复相当于 `2 × 活力层数` 的生命值。 | **可叠加**，提供稳定的回合续航。 |
| **铁壁** (Ironwall) | 面板防御力临时**提升 50%**。 | **可叠加**，最多叠 2 层。 |
| **亢奋** (Excite) | 每次打出卡牌时，有额外的连击能力，会在本回合允许你**额外免费抽牌**（每回合最多触发两次特效补牌）。 | - |
| **反射** (Reflect) | **防守反击**。受到任何实质伤害时，主动**反弹 3 × 层数 点真实伤害给攻击者**。 | **可叠层计算伤害**，通常持续一回合。 |
| **振奋** (Inspire) | **战斗节奏续航**。拥有此增益时，**每打出一张任意卡牌，回复 1 × 层数 点生命**。 | **可无限叠加层数倍率**，此增益一般只在一回合内生效。 |
| **狂暴** (Frenzy) | **战神核心**。每回合内，**每当使用一张攻击牌，必定获得 1 层力量（临时增伤）**。 | **可叠加触发倍率**，此增益一般只在一回合内生效。 |
| **专注** (Focus) | **手牌运转**。每回合**首次使用技能牌时，立刻抽取 1 × 层数 张牌**。 | **可叠加抽牌数量**，此增益通常为一次性触发。 |
| **元素亲和** (Elemental Affinity) | **异常增伤**。敌人因燃烧或中毒受到伤害时，会**额外受到 3 × 层数 的真实伤害**。 | **不可叠加伤害倍率，但可刷新持续回合**。 |
| **连击架势** (Combo Stance) | **物理爆发**。每打出一张攻击牌，**下一张攻击牌的基础伤害增加 2 × 层数 点**。 | **可叠加倍率**，此增益通常仅限本回合行动链。 |
| **时光沙漏** (Extra Turn) | **时停/再动**。回合正常结束后，敌人将永远**失去行动权**，并强制为玩家立刻开启一个新回合。 | **不能叠加。触发即消耗**。 |
