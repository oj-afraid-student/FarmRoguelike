```

  ↓
探索地图
  ↓
进入战斗房间
  ↓
战斗胜利 → 获得物资
  ↓
继续探索
  ↓
完成一层 → 打开末影箱
  ↓

  ↓
进入农场
  ↓
查看作物 → 使用物资加速 → 收获作物
  ↓
应用作物效果
  ↓
继续下一层
```
