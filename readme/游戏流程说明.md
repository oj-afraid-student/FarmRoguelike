# 游戏流程说明

本文档梳理整个游戏的**设计流程**，以及各系统之间的调用关系。

---

## 一、游戏状态（GameState）

| 状态 | 说明 |
|------|------|
| `MainMenu` | 主菜单，开始/加载/退出 |
| `MapExploration` | 地图探索，选择房间移动 |
| `Combat` | 战斗，出牌对战 |
| `Farming` | 农场，种植/收获作物 |
| `RewardSelection` | 奖励选择（战后奖励） |
| `GameOver` | 游戏结束 |

---

## 二、理想流程（设计意图）

```
主菜单 (MainMenu)
    │
    ├─ 开始游戏 ────────────────────────────────────────────────────────────────────────┐
    │                                                                                    │
    ▼                                                                                    │
地图探索 (MapExploration)                                                                │
    │                                                                                    │
    │  MapSystem.GenerateFloor(层数) 生成房间网格                                         │
    │  玩家点击方向按钮 → MapSystem.EnterRoom(新位置)                                    │
    │                                                                                    │
    │  根据房间类型 HandleRoomEnter(room):                                               │
    │  ├─ Combat   → 切换战斗状态，CombatSystem.StartCombat                              │
    │  ├─ Farming  → 切换农场状态，显示 FarmUI                                           │
    │  ├─ Rest     → 恢复/休息                                                           │
    │  ├─ Shop     → 商店                                                                 │
    │  ├─ Event    → 事件                                                                 │
    │  └─ Boss     → Boss 战，胜利后 MapSystem.CompleteCurrentRoom → CompleteFloor       │
    │                                                                                    │
    ▼                                                                                    │
战斗 (Combat)                                                                            │
    │                                                                                    │
    │  玩家出牌 → CombatSystem 处理伤害/防御                                             │
    │  敌人回合 → 攻击玩家                                                               │
    │  敌人死亡 → EmitCombatEnded(true) → CombatResourceSystem 生成物资                 │
    │  玩家死亡 → GameOver(false)                                                        │
    │  战斗胜利 → 返回地图 或 显示奖励选择                                               │
    │                                                                                    │
    ▼                                                                                    │
农场 (Farming)                                                                           │
    │                                                                                    │
    │  点击空地块 → 选择作物 → FarmingSystem.TryPlantCrop                                │
    │  作物生长（真实时间，或物资加速）                                                   │
    │  成熟后收获 → 应用作物效果（CropEffectSystem）                                      │
    │  关闭农场 → 返回 MapExploration                                                    │
    │                                                                                    │
    ▼                                                                                    │
完成一层 (FloorCompleted)                                                                │
    │                                                                                    │
    │  EnderChestSystem 订阅 FloorCompleted                                              │
    │  → 自动 OpenEnderChest(层数)                                                       │
    │  → 显示 EnderChestUI                                                               │
    │  玩家选择作物并支付代价 → 种植到空地块（或加入作物库）                              │
    │  关闭末影箱 → 下一层：MapSystem.GenerateFloor(层数+1)                              │
    │                                                                                    │
    ▼                                                                                    │
循环直到通关或 GameOver ◄───────────────────────────────────────────────────────────────┘
```

## 三、当前实现情况

| 环节 | 状态 | 说明 |
|------|------|------|
| 主菜单 → 开始游戏 | ✅ | `StartNewGame()` → `MapExploration` |
| MapSystem.GenerateFloor | ⚠️ | 需手动调用，`StartNewGame` 未调用 |
| 地图移动 / 进入房间 | ❌ | `StartCombatRoom` 等为 `NotImplementedException` |
| 战斗系统逻辑 | ✅ | CombatSystem 可独立运行 |
| 农场系统 | ✅ | FarmUI 完整，种植/收获/效果 |
| 末影箱 | ✅ | 触发 `FloorCompleted` 即自动打开 |
| 战斗物资 | ✅ | 战斗胜利生成，可对作物使用 |
| 存档/读档 | ✅ | SaveManager 支持 |

## 四、系统依赖关系

```
EventBus（事件总线）
    │
    ├── GameManager ── 状态切换、玩家数据、作物效果接口
    ├── UIManager   ── 根据状态显示对应 UI
    │
    ├── MapSystem        ── 生成地图，房间类型，EmitFloorCompleted
    ├── CombatSystem     ── 战斗逻辑，EmitCombatEnded / EmitEnemyDefeated
    ├── FarmingSystem    ── 6 地块，种植/生长/收获
    ├── CropEffectSystem ── 应用/移除作物效果
    ├── EnderChestSystem ── 订阅 FloorCompleted，打开末影箱
    └── CombatResourceSystem ── 订阅 EnemyDefeated 生成物资，对作物使用
```

---

## 五、如何触发各功能（当前可用的调用路径）

| 功能 | 调用方式 |
|------|----------|
| 进入农场 | `GameManager.ChangeState(GameEnums.GameState.Farming)` |
| 进入战斗 | `CombatSystem.StartCombat("enemy_slime")` + `GameManager.ChangeState(Combat)` |
| 打开末影箱 | `EventBus.EmitFloorCompleted(1)` 或 `GameManager.OpenEnderChest(1)` |
| 种植作物 | `FarmingSystem.TryPlantCrop(plotIndex, cropId)` |
| 收获作物 | `FarmingSystem.TryHarvestCrop(plotIndex)` |
| 应用作物效果 | `GameManager.ApplyCropEffect("crop_health_boost")` |
| 生成战斗物资 | `CombatResourceSystem.GenerateCombatResources(true, "enemy_id")` |
| 使用物资加速 | `GameManager.UseResourceOnCrop(resourceId, plotIndex)` |
| 生成地图 | `MapSystem.GenerateFloor(floorNumber)` |

## 六、全功能测试脚本

项目已添加 `Scripts/Testing/FullGameFlowTest.cs`，用于一次性验证所有系统。

**使用步骤：**

1. 在 Godot 中打开 `Main.tscn`
2. 在根节点 `Main` 下添加一个 `Node` 子节点，命名为 `FullGameFlowTest`
3. 为该节点挂上 `FullGameFlowTest.cs` 脚本
4. 运行游戏，按 **F6** 键执行完整测试

**测试内容：**

- 管理器引用（EventBus、GameManager、DataManager 等）
- DataManager：作物、卡牌、敌人数据
- MapSystem：生成地图
- FarmingSystem：种植作物
- CombatSystem：开启战斗
- CombatResourceSystem：生成战斗物资
- CropEffectSystem：应用作物效果
- EnderChestSystem：打开末影箱
- GameState 切换
- SaveManager 引用

## 七、后续需要补全的流程

1. **游戏启动时调用** `MapSystem.GenerateFloor(1)`
2. **实现** `EventBus.StartCombatRoom`、`StartFarmingRoom` 等，使其切换状态并调用对应系统
3. **地图 UI 的移动按钮** 调用 `MapSystem.EnterRoom`、`GetAvailableMoves` 等
4. **战斗结束后** 正确返回地图或进入末影箱（根据是否为 Boss 房）
