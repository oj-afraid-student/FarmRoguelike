## 🎮 使用流程

### 1. 种植作物流程
```
玩家进入农场 → 点击空地块 → 选择作物 → 种植成功 → 开始生长
```

### 2. 收获作物流程
```
作物成熟 → 点击地块 → 点击收获 → 获得奖励 → 可选择应用效果
```

### 3. 末影箱流程
```
完成一层 → 自动打开末影箱 → 显示作物选项 → 选择作物 → 支付代价 → 自动种植
```

### 4. 使用物资流程
```
战斗胜利 → 获得物资 → 进入农场 → 点击有作物的地块 → 使用物资 → 加速生长
```
