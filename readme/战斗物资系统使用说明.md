# 战斗物资系统使用说明

## 📋 系统概述

战斗物资系统已经完成，实现了以下功能：
1. **战斗胜利自动生成物资** - 击败敌人后自动获得物资
2. **多种物资类型** - 肥料、水、特殊肥料、生长加速剂、品质提升剂
3. **物资对作物的影响** - 加速生长、立即完成进度、增强效果
4. **物资库存管理** - 自动管理玩家拥有的物资

---

## 🎮 使用方法

### 1. 获取物资（自动）

战斗胜利后，系统会自动生成物资并添加到库存：

```csharp
// 系统会自动在EnemyDefeated事件时生成物资
// 也可以通过代码手动生成：
var resourceSystem = GameRoot.Instance.CombatResourceSystem;
var resources = resourceSystem.GenerateCombatResources(true, "enemy_slime");
```

### 2. 查看物资库存

```csharp
var gameManager = GameRoot.Instance.GameManager;

// 获取特定物资数量
int fertilizerCount = gameManager.GetResourceCount("resource_fertilizer");

// 获取所有物资
var allResources = gameManager.GetPlayerResources();
foreach (var resource in allResources)
{
    GD.Print($"{resource.Key}: {resource.Value}");
}
```

### 3. 使用物资到作物

```csharp
// 对地块0使用肥料
bool success = gameManager.UseResourceOnCrop("resource_fertilizer", 0);
if (success)
{
    GD.Print("成功使用肥料，作物生长加速！");
}
```

## 🧪 物资类型详解

### 1. 普通肥料 (resource_fertilizer)
- **效果**: 加速作物生长50%
- **获得概率**: 35%（战斗胜利时）
- **稀有度**: 1（常见）
- **使用场景**: 加速普通作物生长

### 2. 水 (resource_water)
- **效果**: 加速作物生长30%
- **获得概率**: 35%（战斗胜利时）
- **稀有度**: 1（常见）
- **使用场景**: 轻微加速作物生长

### 3. 特殊肥料 (resource_special_fertilizer)
- **效果**: 加速作物生长100%并增强效果20%
- **获得概率**: 5%（战斗胜利时）
- **稀有度**: 3（稀有）
- **使用场景**: 快速成熟并增强作物效果

### 4. 生长加速剂 (resource_growth_boost)
- **效果**: 立即完成作物30%的生长进度
- **获得概率**: 30%（战斗胜利时）
- **稀有度**: 2（不常见）
- **使用场景**: 立即推进作物生长进度

### 5. 品质提升剂 (resource_quality_boost)
- **效果**: 增强作物效果15%
- **获得概率**: 10%（战斗胜利时）
- **稀有度**: 2（不常见）
- **使用场景**: 增强作物效果（收获时应用）

## 📊 物资生成规则

### 基础生成概率

每次战斗胜利时：
- **70%概率** 获得普通肥料或水（二选一）
- **30%概率** 获得生长加速剂
- **10%概率** 获得品质提升剂
- **5%概率** 获得特殊肥料（稀有）

### 生成示例

假设战斗胜利10次：
- 约7次获得肥料/水
- 约3次获得生长加速剂
- 约1次获得品质提升剂
- 约0.5次获得特殊肥料（可能0次或1次）

## 🔧 系统架构

### 核心类

1. **CombatResourceSystem** (`Scripts/Managers/CombatResourceSystem.cs`)
   - 管理物资生成和库存
   - 处理物资对作物的影响
   - 与FarmingSystem集成

2. **ResourceData** (`Scripts/DataTypes/CombatResourceData.cs`)
   - 定义物资数据
   - 包含效果参数和稀有度

3. **ResourceType** (`Scripts/DataTypes/CombatResourceData.cs`)
   - 物资类型枚举
   - 5种不同类型的物资

### 事件系统

系统通过EventBus发送以下事件：
- `CombatResourcesGenerated` - 物资生成时触发
- `ResourceAppliedToCrop` - 物资应用到作物时触发
- `CombatEnded` - 战斗结束时触发（用于生成物资）

### 集成点

- **GameRoot**: 注册了CombatResourceSystem单例
- **GameManager**: 提供了便捷的接口方法
- **CombatSystem**: 在EnemyDefeated时触发物资生成
- **FarmingSystem**: 接收物资效果，加速作物生长

## 💡 使用建议

### 在UI中显示物资

```csharp
// 显示玩家物资库存
void UpdateResourceUI()
{
    var resources = gameManager.GetPlayerResources();
    
    foreach (var resource in resources)
    {
        var resourceData = resourceSystem.GetResourceData(resource.Key);
        if (resourceData != null)
        {
            // 创建UI元素
            CreateResourceUIElement(resourceData, resource.Value);
        }
    }
}
```

### 在农场UI中使用物资

```csharp
// 玩家点击地块，显示使用物资选项
void OnPlotClicked(int plotIndex)
{
    var plots = farmingSystem.Plots;
    if (plots[plotIndex].IsOccupied)
    {
        // 显示可用物资列表
        ShowResourceSelection(plotIndex);
    }
}

void ShowResourceSelection(int plotIndex)
{
    var resources = gameManager.GetPlayerResources();
    
    foreach (var resource in resources)
    {
        if (resource.Value > 0)
        {
            // 创建使用按钮
            CreateUseResourceButton(resource.Key, plotIndex, () => {
                gameManager.UseResourceOnCrop(resource.Key, plotIndex);
            });
        }
    }
}
```

### 物资效果提示

```csharp
string GetResourceEffectDescription(string resourceId)
{
    var resourceData = resourceSystem.GetResourceData(resourceId);
    if (resourceData == null)
        return "";
    
    switch (resourceData.Type)
    {
        case ResourceType.Fertilizer:
            return $"加速生长 {resourceData.GrowthAcceleration * 100}%";
        case ResourceType.Water:
            return $"加速生长 {resourceData.GrowthAcceleration * 100}%";
        case ResourceType.SpecialFertilizer:
            return $"加速生长 {resourceData.GrowthAcceleration * 100}%，增强效果 {resourceData.EffectEnhancement * 100}%";
        case ResourceType.GrowthBoost:
            return $"立即完成 {resourceData.InstantProgress * 100}% 进度";
        case ResourceType.QualityBoost:
            return $"增强效果 {resourceData.EffectEnhancement * 100}%";
        default:
            return resourceData.Description;
    }
}
```

## 🐛 注意事项

1. **地块检查**: 使用物资前会检查地块是否有作物
2. **物资消耗**: 使用物资后会从库存中扣除
3. **效果叠加**: 可以对同一作物多次使用物资，效果会叠加
4. **立即效果**: 生长加速和立即进度会立即生效
5. **品质提升**: 品质提升剂的效果在收获时应用（需要扩展CropReward系统）

## 📝 示例流程

1. 玩家击败敌人
2. CombatSystem触发EnemyDefeated事件
3. CombatResourceSystem自动生成物资（如：普通肥料 x1）
4. 物资添加到玩家库存
5. 玩家进入农场，看到地块0有作物
6. 玩家选择使用"普通肥料"到地块0
7. 系统检查物资库存和地块状态
8. 应用加速效果：将作物种植时间提前50%
9. 消耗1个肥料
10. 作物生长进度立即更新

2. **重要作物**: 使用特殊肥料快速成熟并增强效果

4. **效果强化**: 使用品质提升剂增强作物效果
