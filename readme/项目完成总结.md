## 🎯 核心功能说明

### 1. 作物系统
- **种植**: 6个地块，点击空地块选择作物
- **生长**: 根据现实时间自动生长
- **收获**: 成熟后点击收获，获得奖励
- **效果**: 收获后可选择应用效果

### 2. 末影箱系统
- **触发**: 每层完成自动打开
- **选择**: 3-5个随机作物选项
- **代价**: 金币/生命/卡牌三种类型
- **种植**: 选择后自动种植到空余地块

### 3. 战斗物资系统
- **生成**: 战斗胜利按概率生成
- **类型**: 5种物资（肥料、水、特殊肥料等）
- **使用**: 对作物使用加速生长
- **库存**: 自动管理玩家物资

### 4. 数据持久化
- **保存**: 保存所有作物状态、物资、效果
- **加载**: 恢复所有数据并计算离线生长
- **快速保存**: 支持快速存档功能
