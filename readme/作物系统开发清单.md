# 作物系统开发清单

## 📋 概述
根据游戏需求，作物系统需要实现以下核心功能：
1. **作物作为商店替代** - 提供属性加成、诅咒权衡、遗忘等效果
2. **末影箱系统** - 每层结束通过末影箱获得作物（需付出代价）
3. **现实时间生长** - 作物根据现实时间生长
4. **战斗物资影响** - 战斗结束的物资可以加速作物生长

---

- ✅ `FarmingSystem.cs` - 基础种植、收获、生长更新
- ✅ `CropData` / `CropReward` - 数据类定义

## 🎯 需要完成的代码模块

### 1. **作物效果系统** (CropEffectSystem.cs)
**位置**: `Scripts/Managers/CropEffectSystem.cs`  
**功能**:
- 实现作物的三种效果类型：
  - **属性加成**：提高生命上限等
  - **诅咒权衡**：降低速度20%，提高攻击力30%
  - **遗忘**：删除卡牌或属性
- 管理已激活的作物效果
- 应用/移除作物效果到玩家数据

**关键方法**:
```csharp
- ApplyCropEffect(CropData crop) // 应用作物效果
- RemoveCropEffect(string cropId) // 移除作物效果
- GetActiveEffects() // 获取当前激活的效果列表
```

---

### 2. **末影箱系统** (EnderChestSystem.cs)
**位置**: `Scripts/Managers/EnderChestSystem.cs`  
**功能**:
- 每层结束时提供作物选择
- 玩家需要付出代价（金币/生命/卡牌等）才能获得作物
- 随机生成可选择的作物列表
- 处理作物选择逻辑

**关键方法**:
```csharp
- OpenEnderChest(int floorNumber) // 打开末影箱，生成可选作物
- SelectCrop(string cropId, int cost) // 选择作物并扣除代价
- GetAvailableCrops(int floorNumber) // 根据层数获取可用作物
```

**相关事件**:
- 需要在 `EventBus` 中添加：
  - `EnderChestOpened` - 末影箱打开
  - `CropSelectedFromChest` - 从末影箱选择作物

---

### 3. **战斗物资系统** (CombatResourceSystem.cs)
**位置**: `Scripts/Managers/CombatResourceSystem.cs`  
**功能**:
- 战斗结束后生成物资（肥料、水等）
- 物资可以加速作物生长
- 物资可以影响作物效果
- 管理玩家拥有的物资

**关键方法**:
```csharp
- GenerateCombatResources(bool isVictory) // 战斗结束生成物资
- ApplyResourceToCrop(int plotIndex, string resourceId) // 对作物使用物资
- GetPlayerResources() // 获取玩家拥有的物资
```

**物资类型**:
- 肥料：加速生长50%
- 水：加速生长30%
- 特殊肥料：加速生长100%并增强效果

## 🔗 与其他系统的接口

### 与战斗系统的接口
- 战斗结束 → 生成物资 → 可用于加速作物
- 战斗胜利 → 解锁珍稀作物

### 与地图系统的接口
- 每层完成 → 触发末影箱 → 选择作物

### 与玩家数据的接口
- 作物效果 → 修改玩家属性
- 收获作物 → 获得奖励（生命、金币等）

## 📌 注意事项

1. **现实时间计算**：使用 `DateTime.Now` 计算离线生长时间
2. **效果叠加**：考虑多个作物效果的叠加规则
3. **代价平衡**：末影箱的代价需要合理设计
4. **数据持久化**：确保作物状态在游戏重启后能恢复
5. **性能优化**：作物更新使用定时器，避免每帧检查
